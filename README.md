# Jayden's Interactive CV Website 🚀

A stunning, animation-heavy CV website built with modern web technologies.

## 🎨 Features

- **Advanced Animations**: Powered by Framer Motion and custom CSS animations
- **3D Graphics**: Interactive 3D elements using Three.js and React Three Fiber
- **Smooth Interactions**: Buttery-smooth page transitions and micro-interactions
- **Particle System**: Custom particle background effect
- **Responsive Design**: Looks amazing on all devices
- **Performance Optimized**: Fast loading and smooth 60fps animations
- **GitHub Pages Ready**: Easy deployment to GitHub Pages

## 🛠️ Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Graphics**: Three.js, React Three Fiber, Drei
- **Deployment**: GitHub Pages

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/JaydeninjaDevWebsiteCV.git
cd JaydeninjaDevWebsiteCV
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## 📦 Building for Production

Build the project:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## 🌐 Deploying to GitHub Pages

1. Update the `base` path in `vite.config.js` with your repository name
2. Push your code to GitHub
3. Run the deployment command:
```bash
npm run deploy
```

Your site will be live at `https://yourusername.github.io/JaydeninjaDevWebsiteCV/`

## 📝 Customization

### Update Personal Information

Edit the following files to customize with your information:
- `src/components/Hero.jsx` - Name and intro
- `src/components/About.jsx` - About section
- `src/components/Skills.jsx` - Your skills
- `src/components/Projects.jsx` - Your projects
- `src/components/Contact.jsx` - Contact information

### Styling

- Colors and theme: `tailwind.config.js`
- Global styles: `src/index.css`
- Component-specific styles: Use Tailwind classes in components

## 🎯 Performance Tips

- Images: Optimize images before adding them
- 3D Models: Keep polygon count reasonable
- Animations: Use `transform` and `opacity` for best performance
- Lazy Loading: Consider lazy loading heavy components
